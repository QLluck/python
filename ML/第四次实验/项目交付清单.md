## 🚀 使用方法

### 方法1：Jupyter Notebook（推荐）
```bash
# 1. 安装依赖
pip install -r requirements.txt

# 2. 启动Jupyter
jupyter notebook

# 3. 打开SVM实验.ipynb并运行所有单元格
```

### 方法2：Python脚本
```bash
# 运行完整实验
python3 main.py

# 或分步运行
python3 svm_experiment_part1.py  # 数据准备
```

1. **中文字体**：Mac使用'Arial Unicode MS'，Windows使用'SimHei'
2. **随机种子**：所有代码设置random_state=42，确保可复现
3. **文件路径**：使用相对路径，确保跨平台兼容
