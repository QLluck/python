## 🤖 智能模式（Smart Mode）使用指南

智能模式使用 AI 自动预测最优的分割参数，适合快速处理和不熟悉参数调整的用户。

### 步骤 1: 切换到智能模式

在页面顶部找到三个模式标签，点击 **"智能模式"**：

```
┌─────────────────────────────────────────┐
│ [手动模式] [智能模式✓] [点击模式]        │
└─────────────────────────────────────────┘
```

### 步骤 2: 上传图像

1. 点击上传区域或拖拽图像文件
2. 支持格式：PNG、JPG、BMP
3. 建议尺寸：512x512 到 2048x2048

### 步骤 3: 预测参数

点击 **"预测最优参数"** 按钮：

```
┌─────────────────────────────────────────┐
│  🤖 预测最优参数                         │
└─────────────────────────────────────────┘
```

系统会：
- 提取图像的 28 个特征
- 使用 ML 模型预测最佳分割方法
- 预测该方法的最优参数
- 显示预测置信度

### 步骤 4: 查看预测结果

预测完成后会显示：

```
┌─────────────────────────────────────────┐
│ AI 预测结果              置信度: 87.3%  │
├─────────────────────────────────────────┤
│ 推荐方法: 区域生长                       │
│                                         │
│ 参数:                                   │
│   grow_T: 18.5                         │
│   seed_strategy: center                │
│   connectivity: 8                      │
│                                         │
│ 💡 您可以在下方手动调整预测的参数        │
└─────────────────────────────────────────┘
```

**置信度颜色说明：**
- 🟢 绿色 (>80%): 高置信度，可直接使用
- 🟡 黄色 (50-80%): 中等置信度，建议微调
- 🔴 红色 (<50%): 低置信度，建议手动调整

### 步骤 5: 调整参数（可选）

预测后，参数控制区域会自动显示并预填充预测值。你可以：
- 直接使用预测的参数
- 手动微调任何参数
- 展开高级选项进行精细调整

### 步骤 6: 运行处理

点击 **"运行处理"** 按钮开始分割。

## 👆 点击模式（Click Mode）使用指南

点击模式让你直接在图像上点击病灶位置，系统自动进行分割，无需调整参数。

### 步骤 1: 切换到点击模式

点击 **"点击模式"** 标签：

```
┌─────────────────────────────────────────┐
│ [手动模式] [智能模式] [点击模式✓]        │
└─────────────────────────────────────────┘
```

### 步骤 2: 上传图像

1. 上传要处理的医学图像
2. 图像会显示在结果区域

### 步骤 3: 点击病灶位置

**在图像上直接点击病灶的中心位置：**

```
        图像预览
┌─────────────────────────┐
│                         │
│         ●               │  ← 点击这里
│      病灶区域            │
│                         │
└─────────────────────────┘
```

系统会：
1. 提取点击位置周围的局部特征
2. 使用 ML 预测该区域的最优 grow_T 参数
3. 从点击点开始进行区域生长分割
4. 实时显示分割结果（红色覆盖层）

### 步骤 4: 多次点击优化（可选）

如果单次点击效果不理想，可以：

1. **添加更多点击点**
   - 在其他病灶区域点击
   - 系统会合并所有点击的结果

2. **查看点击计数**
   ```
   点击次数: 3
   ```

3. **清除重来**
   - 点击 **"清除点击"** 按钮
   - 重新开始点击

### 步骤 5: 查看结果

分割完成后会显示：
- 原始图像
- 分割掩膜（黑白）
- 叠加效果（红色高亮）

## 💡 使用技巧

### 智能模式技巧

1. **置信度判断**
   - 高置信度（>80%）：直接使用预测参数
   - 中等置信度（50-80%）：查看预测，可能需要微调
   - 低置信度（<50%）：建议切换到手动模式

2. **参数微调**
   - 如果分割过度：减小 `grow_T` 值
   - 如果分割不足：增大 `grow_T` 值
   - 如果边界不平滑：调整形态学参数

3. **图像质量**
   - 清晰的图像 → 更高的预测准确度
   - 模糊或低对比度图像 → 可能需要手动调整

### 点击模式技巧

1. **点击位置**
   - ✅ 点击病灶的中心区域
   - ✅ 选择颜色/亮度最典型的位置
   - ❌ 避免点击边缘或模糊区域

2. **多次点击策略**
   - 第一次点击：病灶主体
   - 第二次点击：未覆盖的区域
   - 第三次点击：细节补充

3. **结果不理想时**
   - 点击 "清除点击" 重新开始
   - 尝试不同的点击位置
   - 或切换到智能模式进行参数调整

## 🔧 常见问题

### Q1: 智能模式预测失败怎么办？

**A:** 可能原因：
1. 模型未训练 → 运行 `python scripts/train_model.py`
2. 图像质量差 → 尝试预处理或使用更清晰的图像
3. 图像类型不匹配 → 系统会使用启发式规则作为后备

### Q2: 点击模式没有反应？

**A:** 检查：
1. 是否已上传图像
2. 是否点击在图像区域内
3. 查看浏览器控制台是否有错误
4. 刷新页面重试

### Q3: 如何提高预测准确度？

**A:** 
1. 训练模型时使用更多样本
2. 收集真实用户数据进行重新训练
3. 使用高质量的训练图像

### Q4: 三种模式可以混合使用吗？

**A:** 可以！推荐工作流程：
1. 先用 **点击模式** 快速尝试
2. 如果效果不理想，切换到 **智能模式** 查看 AI 建议
3. 如果还需要优化，切换到 **手动模式** 精细调整

## 📝 实际操作示例

