# SVM实验快速开始指南

## 方式一：使用Jupyter Notebook（推荐）

### 步骤1：安装依赖
```bash
pip install -r requirements.txt
```

### 步骤2：启动Jupyter Notebook
```bash
jupyter notebook
```

### 步骤3：打开并运行
1. 在浏览器中打开 `SVM实验.ipynb`
2. 依次运行每个单元格（Shift + Enter）
3. 查看实验结果和可视化图表
4. 最后一个单元格会自动生成Word报告

## 方式二：直接运行Python脚本

### 运行完整实验
```bash
python3 main.py
```

这将自动完成：
- 数据加载与可视化
- 模型训练与评估
- 超参数调优
- 生成所有图表（保存在 `figures/` 文件夹）
- 生成Word报告（`SVM实验报告.docx`）

### 分步运行
```bash
# 第一步：数据准备
python3 svm_experiment_part1.py

# 第二步：模型训练（需要先运行第一步）
python3 svm_experiment_part2.py

# 第三步：超参数调优（需要先运行前两步）
python3 svm_experiment_part3.py

# 生成Word报告
python3 generate_report.py
```

## 方式三：仅生成Word报告

如果已经运行过实验并生成了图片，可以直接生成报告：

```bash
python3 generate_report.py
```

## 常见问题

### Q1: 中文显示为方框
**A**: 修改字体设置，在代码中找到：
```python
plt.rcParams['font.sans-serif'] = ['Arial Unicode MS', 'SimHei']
```
根据你的系统选择合适的中文字体。

### Q2: 导入模块失败
**A**: 确保所有.py文件在同一目录下，或者使用：
```python
import sys
sys.path.append('.')
```

### Q3: 图片不显示
**A**: 
- Jupyter中：确保运行了 `%matplotlib inline`
- Python脚本：图片会自动保存到 `figures/` 文件夹

### Q4: Word报告生成失败
**A**: 确保安装了 `python-docx`：
```bash
pip install python-docx
```
