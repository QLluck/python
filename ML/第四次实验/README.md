# SVM实验作业 - 支持向量机分类与回归实验

## 项目简介
本项目是一个完整的支持向量机（SVM）机器学习实验，包含数据准备、模型训练、参数调优、结果可视化和报告生成。

## 实验环境要求
- Python 3.7+
- Jupyter Notebook

## 依赖库安装
```bash
pip install numpy pandas matplotlib seaborn scikit-learn python-docx imbalanced-learn jupyter
```

## 项目结构
```
第四次实验/
├── README.md                    # 项目说明文档
├── requirements.txt             # 依赖库列表
├── SVM实验.ipynb                # 主实验Notebook
├── generate_report.py           # 生成Word报告的脚本
├── data/                        # 数据文件夹（自动生成）
├── figures/                     # 图片保存文件夹（自动生成）
└── SVM实验报告.docx             # 生成的实验报告（运行后生成）
```

## 使用方法

### 方法一：使用Jupyter Notebook（推荐）
1. 启动Jupyter Notebook
   ```bash
   jupyter notebook
   ```
2. 打开 `SVM实验.ipynb`
3. 按顺序运行所有单元格
4. 实验完成后会自动生成Word报告

### 方法二：直接运行报告生成脚本
```bash
python generate_report.py
```

## 实验内容

### 1. 数据集
- **数据集1**：线性可分数据集（Iris鸢尾花数据集）
- **数据集2**：非线性可分数据集（乳腺癌数据集）

### 2. 实验步骤
1. 数据加载与探索性分析
2. 数据预处理（标准化、划分训练测试集）
3. 线性核SVM模型训练
4. 多项式核SVM模型训练
5. 高斯核（RBF）SVM模型训练
6. 超参数调优（网格搜索）
7. 模型性能评估与对比
8. 结果可视化
9. 处理非平衡数据（可选）
10. 生成实验报告

### 3. 可视化内容
- 原始数据分布图
- 不同核函数的决策边界图
- 支持向量可视化
- 混淆矩阵热力图
- 参数调优结果图
- 模型性能对比图

## 实验报告
运行完成后会自动生成 `SVM实验报告.docx`，包含：
- 第一部分：算法描述、数据集介绍、实验方案
- 第二部分：程序清单（完整代码+注释）
- 第三部分：实验结果（图表和分析）
- 第四部分：疑难小结（问题与心得）

## 注意事项
- 所有代码都设置了随机种子（random_state=42），确保结果可复现
- 代码风格简洁易懂，适合大学生学习
- 每段代码都有详细的中文注释
- 图片会自动保存到 `figures/` 文件夹
