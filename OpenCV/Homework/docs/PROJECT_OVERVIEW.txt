📦 核心功能

  ✅ 图像输入与管理        支持 PNG/JPG/BMP，自动缩放大图
  ✅ 预处理               灰度化、中值、双边、CLAHE、形态学
  ✅ 病变候选检测          ROI 检测，支持灰度/彩色模式
  ✅ 精确分割             Otsu / 区域生长 / 分水岭（3种方法）
  ✅ 后处理               孔洞填充、小区域去除
  ✅ Web 系统界面         FastAPI + 静态前端
  ✅ 评估与可视化          Dice/IoU、LBP 纹理、轮廓叠加
