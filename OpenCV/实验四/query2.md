## 实验1：图像分割

对 **图1、图2、图3** 三幅图像，各选择 **两种不同类型** 的分割算法（阈值法、边缘法、区域法各选其一组合），输出分割结果图。

具体要求：

1. **图1（低对比度医学图像）**：使用 **自适应阈值法** + **Canny边缘检测**。先用自适应阈值做初步分割，再用 Canny 提取边缘叠加显示。
2. **图2（自然场景图像）**：使用 **Otsu全局阈值法** + **分水岭算法**。先转灰度后用 Otsu 二值化，再基于距离变换做分水岭分割。
3. **图3（不均衡曝光图像）**：使用 **局部自适应阈值** + **Sobel边缘检测**。对曝光不均区域用自适应阈值处理，再用 Sobel 算子检测边缘。
4. 每幅图用 `plt.subplots` 并排显示原图和两种分割结果，标题注明算法名称。
5. 在 Markdown 单元格中简要说明算法选择依据，分析图像特性与算法适应性之间的关系，并对比两种算法的优缺点。

## 实验3：局部特征提取

### 任务3.1：SIFT 特征匹配

使用 **图4（标准Lena）** 和 **图5（旋转缩放后的Lena）** 进行 SIFT 特征匹配。

具体要求：

1. 使用 `cv2.SIFT_create()` 提取两幅图的 SIFT 关键点和描述符。
2. 使用 **暴力匹配（Brute-Force）** `cv2.BFMatcher` 进行匹配。
3. 筛选 **最近邻距离比（NNDR）< 0.7** 的匹配对（使用 `knnMatch`，k=2，应用 Lowe's ratio test）。
4. 使用 `cv2.drawMatches` 绘制匹配连线图，保存为 `实验3_SIFT匹配连线图.png`。
5. 计算并输出匹配正确率：使用 `cv2.findHomography` + RANSAC 计算内点数量，内点数 / 筛选后匹配数 = 正确率。

### 任务3.2：Harris 角点检测

使用 **图4（标准Lena）** 进行 Harris 角点检测。

具体要求：

1. 使用 `cv2.cornerHarris` 实现 Harris 角点检测。
2. 分别设置 **3 种不同阈值**（如 `0.01 * dst.max()`, `0.05 * dst.max()`, `0.1 * dst.max()`），用 `plt.subplots` 并排显示三种阈值下的角点检测结果，标题注明阈值和角点数量。
3. 分析不同阈值对角点数量和分布的影响。
4. 在 Markdown 单元格中对比 Harris 与 SIFT 特征点的分布差异，分析二者在特征点检测中的表现（尺度不变性、旋转不变性、稠密程度等）。
